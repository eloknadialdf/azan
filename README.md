# azan

Indonesia Azan clock

## Installation
```
npm install -g azan
```

## Usage

```
$ azan
```
Print monthly prayer times.
```
$ azan --monthly
```

## Configuration

```
$ azan --city=CITY_ID
```
Available city
```
$ azan --help
```
